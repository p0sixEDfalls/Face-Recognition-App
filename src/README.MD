# Face-Recognition-App

Весь код в папке src. Каждый файл представляет собой модуль:
1. collect-faces.py - открывает окно и начинает распознавать лица, нажимаешь кнопку s и твое лицо сохраняется в папку /data/train/Nikita с нумерацией
2. shared.py - содержит общие данные для всех модулей
3. reorganization_unknow_data.py - берет фотки из папки /data/originals, обрезает лица, нумеруют и кладет в /data/train/Unknow
4. create_test_data.py - берет процент из папок с трэйнингом и кладет в соответствующие папки в /data/test/..
5. testing.py - распознает заданное изображение и выводит результат в консоль
6. model.py - содержит описание обучение и сохрание модели.
7. prediction.py - содержит функцию, возвращающая имя класса, определенного на входных данных.
